.NET Diploma

(Diploma in .Net)




Objective

At the end of this course, the candidate will acquire a deep knowledge in programming, object-oriented model, web programming, and enterprise application development.

Target Audience

A fresh graduate engineer, MCA, M.Sc., B.Sc., BCA, software professional.

Prerequisites

Basic programming knowledge is required.

Overview

This course is designed to create a new tool in software development using an object-oriented environment, which will step into normal programming, web technology, and enterprise application. This course is for those who want to become a software developer in the profession.

Project

Students will develop web-based applications like online reservation systems, online share trading, live websites, etc. Academic projects completed in this course can be used as final-year college projects.
