ADPP

(Advanced Diploma in Python Programming)




Objective

Upon completion of this course, the participant will be in a position to create components for the internet and integrate them.

Target Audience

Diploma/Graduates/Engineering Students

Prerequisites

Basic knowledge of computing

Overview

This course enables participants with skills to do effective programming for internet applications. It teaches advanced programming techniques for the internet.
