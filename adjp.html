ADJP

(Advanced Diploma in Java Programming)




Objective

Upon completion of this course, the participant will be in a position to create components for the web and integrate them.

Target Audience

Diploma/Graduates/Engineering Students

Prerequisites

Basic knowledge of computing

Overview

The course enables you with effective programming for internet applications. It teaches you advanced programming techniques for the development of web-based applications.
