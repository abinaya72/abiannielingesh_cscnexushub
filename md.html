MDSA

(Master Diploma in Hardware & Networking)




Objective

Upon completion of this course, one can confidently obtain international certifications like A+ and N+. Immediately after finishing this course, you will be well-versed in Hardware & System Administration in Networking.

Target Audience

Candidates with a minimum of 10th grade, Diploma Degree in Engineering, MCA/BCA, or any software professional or technician in electronics.

Prerequisites

No prior computer knowledge is required.

Overview

The Hardware & Networking Course is taught from scratch, covering the subject in detail.

Course Coverage

  • Hardware (A+)

    • Introduction to Computers
    • System Familiarization
    • Software Classification
    • Commands
    • Processor & Motherboard Details
    • BIOS Setup
    • Storage Devices (Primary & Secondary)
    • Virus & Antivirus Concepts
    • Power Section
    • Assembling and Installation of OS
    • OS Properties
    • Input & Output Devices
    • Troubleshooting Tips

  • Networking (N+)

    • Introduction to Networks
    • Network Essentials
    • Topology
    • Connecting Devices
    • OSI Model
    • TCP/IP Overview
    • Installation of OS (Windows) & NOS (Windows Server)
    • Connectivity and Resource Sharing
    • Remote Tools
    • DHCP Server Configuration
    • File System & Disk Management
    • Troubleshooting Tips

  • Windows Server

    • Deploying and Managing Windows Server
    • Introduction to Active Directory Domain Services
    • Managing Active Directory Domain Services Objects
    • Implementing IPv4
    • Implementing Dynamic Host Configuration Protocol
    • Implementing Domain to System
    • Implementing Local Storage
    • Implementing File and Print Services
    • Implementing Server Virtualization with Hyper-V
