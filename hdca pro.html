HDCA

(Honours Diploma in Computer Application)




Objective

Upon completion of this course, the participant will acquire the advantage of choosing a career as an Application Programmer, creating applications using object-oriented programming concepts such as inheritance, polymorphism, and virtual functions.

Target Audience

School final students, Graduates, Students pursuing a career in Computer Science.

Prerequisites

No prior computer knowledge is required.

Overview

This course is tailor-made according to software industry requirements. It is a stepping stone for those aspiring to become software professionals.
