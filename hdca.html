HDCA

(Honours Diploma in Computer Application)




Objective

Upon completion of this course, the participant gains expertise in Tally Prime, VBA Excel, and other essential software tools, preparing them for careers as Application Programmers, Computerized Accountants, and Web Programmers.

Target Audience

School final students, Graduates, or students pursuing a career in computers.

Prerequisites

No prior computer knowledge is required.

Overview

This course is tailored to meet software industry requirements and serves as a stepping stone for those aspiring to be software professionals.

Course Coverage

  • End-User Applications

    • Windows: Computer Fundamentals, Introduction to OS/GUI, Wordpad, Notepad, Paintbrush, Windows Explorer, Control Panel.
    • MS Office: Word - Creating Text and Documents, Indexes, Tables, Embedding Pictures, Table Manipulation, Mail Merge, Macro. Excel - Formatting Cells, Charts, Pivot Tables, Data Table, Forms, Scenarios, Goal Seek, Macros, and Module. PowerPoint - Creating Presentations, Enhancing Presentations, Adding Audio & Video, Special Effects.
    • Internet: Creating E-mail ID, Receiving & Sending E-mails, Chatting, Locating Information, Downloading.

  • Front-End Language & Library

    • HTML5: HTML Basics, New Elements, Canvas, Multimedia, Form Elements, Semantic Elements.
    • CSS3: Syntax, Id, Class, Tag Styling, Using Text Fonts, Links, Lists and Tables, Box Model, Text Effects, Transition, Animations.

  • Database Management System

    • MySQL: Client/Server Concepts, Relational Model, Data Definition Using SQL, Modify Tables Using ALTER TABLE Queries, Insert and Delete Queries, Select Queries, Grouping, Functions, Math Functions, String Comparisons, Work with Dates, JOIN and Subqueries.

  • Financial Accounting

    • Introduction to TallyPrime: Principles of Accounting - 5 Steps, Maintaining Company Data, F11 Features, F12 Configure.
    • Masters: Group, Ledger, Chart of Accounts, Stock Group, Stock Category, Unit of Measurement, Godown, Stock Item.
    • Tally Vouchers: Accounting, Inventory Vouchers.
    • Reports: Balance Sheet, Profit & Loss Account, Ratio Analysis, Cash Flow, Fund Flow Statements, Ageing Analysis, Movement Analysis.
    • Advanced Accounting: Billwise Details, Cost Centres and Cost Categories, Voucher Classes, Multiple Currencies, Bank Reconciliation, Interest Calculation, Budgets & Scenario, Job Costing.
    • Advanced Inventory: Order Processing, Reorder Levels, Tracking Numbers, Batchwise Details, Additional Cost Details, Bill of Materials, Price List.

  • System Administration and Other Utilities

    • Consolidation of Accounts: Tally Vault Security Control, Tally Audit, Backup and Restore, Split Company Data, Export and Import of Data, ODBC Compliance.
    • Point of Sales: Predetermination of Invoice, Multi Payment Modes, Small & Smart Bills, POS Sales Register.
    • Multi Language: Supporting OS Translation, Transliteration System, Phonetic Keyboard, Sourcing from Other Software.
    • TDS & TCS: Entry, Calculator Reports, E-Returns.
    • Payroll: Introduction to Payroll, Configuring Tally for Payroll, Creation of Payroll Masters, Recording Attendance, Preparation of Payroll Vouchers, Salary Disbursement Voucher, Payroll Reports (Pay Slip, Pay Sheet, Payroll Statement, Payment Advice, Payroll Register, Attendance Sheet, Attendance Register), Introduction to Provident Fund.

  • Advanced Excel & VBA Programming

    • Advanced Excel: Working with Formulas and Functions (Lookup and Reference Functions, Logical Functions, Database Functions, Date and Time Functions, Math and Trigonometry Functions, Statistical Functions, Text Functions), Conditional Formatting, Data Sorting and Filtering, Pivot Tables & Charts.
    • VBA Programming: Overview, Variables, Conditional Logic, String and String Functions, Programming Loops, Arrays, Subs and Functions, Creating Macros, Excel VBA Objects and Events, Error Handling, User Forms, External Data Connections, Automation Projects.

  • Goods & Service Tax (GST)

    • Accounting of Purchase/Sales Returns, Inter-State Trade Accounting, Exempted Items, Purchases from Unregistered Dealers, Accounting Input Credit on Capital Goods, GST Computation & Forms.

  • Digital Marketing

    • Introduction to Digital Marketing, Traditional vs Digital Marketing, Organic vs Inorganic Marketing, Content Creation, Graphic Designing, Social Media Marketing, Facebook, Instagram, YouTube, Search Marketing, Google Ads, Google Analytics.
