CCAE

(Certificate Course in Advanced Excel)




Objective

Upon completion of this course, the participant will be an expert in advanced Excel features, including macro programming and functions.

Target Audience

BBA, MBA students, Data Analysts, Auditors, BPO/Call Center professionals, Data Conversion Technicians, and Programmers.

Overview

This course helps in promoting Excel proficiency at an advanced level.

Course Coverage

  • Advanced Excel

    • Introduction to MS Excel
    • Understanding Basic Functions
    • Formatting Worksheets
    • Working with Formulas/Functions (Lookup and Reference Functions, Logical Functions, Database Functions, Date and Time Functions, Information Functions, Math and Trigonometry Functions, Statistical Functions, Text Functions)
    • Conditional Formatting
    • Connecting to Other Databases
    • Data Sorting and Filtering
    • Pivot Tables
    • Charts

  • VBA Programming

    • VBA Overview
    • VBA Programming Variables
    • Conditional Logic
    • String and String Functions
    • Programming Loops
    • Programming Arrays
    • Subs and Functions
    • Creating Macros
    • Excel VBA Objects
    • Excel Events
    • VBA Error Handling
    • User Forms
    • Connecting with External Data
    • Project work
