DFJD

(Diploma in Full Stack Java Developer)




Objective

This course helps participants master both front-end and back-end Java technologies, accelerating their career as a full-stack software developer.

Target Audience

Diploma/Graduates/B.E. Students.

Prerequisites

Basic knowledge of computing is required.

Overview

This course provides participants with skills in both front-end and back-end development, covering Java programming, web technologies, database management, and frameworks.

Course Coverage

  • Front-End Language & Library

    • HTML5: Basics, New Elements, Semantic Elements, Canvas, Multimedia, Forms, Promises, Async/Await
    • CSS3: Syntax, Class & Tag Styling, Text Fonts, Links, Lists & Tables, Box Model, Text Effects, Transitions, Animations
    • Bootstrap: Responsive Layouts with Grids, Themes, Components, Building Forms
    • Advanced JavaScript: Basics, Arrays, Functions, DOM Model, Event Handling, Arrow Functions, Closures, Array Filter & Reduce, Regular Expressions, OOPs
    • React JS: Creating an App, Using Templates, Composing with Props and State, Event Handling, useState, useEffect, useRef, useCallback, Server Integration with API Calls

  • Core Language

    • Core Java: Data Types, Arrays, Classes, Inheritance, Polymorphism, Abstraction, Packages, Exceptions, File Streams

  • Database Management System

    • MySQL: Client/Server Concepts, Relational Model, SQL Data Definition, Modify Tables, Queries (INSERT, DELETE, UPDATE), SELECT Queries, JOIN, Constraints, Indexes, Views, Advanced MySQL Query Processing
    • MongoDB: Introduction to NoSQL Databases, CRUD Operations, Data Indexing & Aggregation, Working with Unstructured Data

  • Web Framework

    • Spring Boot: Annotations, CRUD Operations, Spring Data JPA, Custom Queries in JPA, Hibernate, Java REST API, Spring Boot with React Integration
