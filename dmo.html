DMO

(Diploma in MS Office)




Objective

Upon completion of this course, the participant will be proficient in handling office tasks efficiently using MS Office applications.

Target Audience

Secretaries, Administrative Professionals, and Office Assistants.

Prerequisites

No prior experience with MS Office is required. Basic knowledge of computers is sufficient.

Overview

This course covers the complete MS Office suite, preparing participants for documentation, data management, presentations, and other office-related tasks using computers.

Course Coverage

  • Windows

    • Computer Fundamentals
    • Introduction to OS
    • GUI
    • WordPad
    • Notepad
    • Paintbrush
    • Windows Explorer
    • Control Panel

  • MS Office

    • Word: Creating Text and Documents, Indents & Tables, Embedding Pictures, Table Manipulation, Mail Merge, Macros.
    • Excel: Formatting Cells, Charts, Pivot Table, Data Tables, Formulas, Scenarios, Goal Seek, Macros.
    • PowerPoint: Creating Presentations, Enhancing Presentations, Adding Audio & Video, Special Effects.

  • Internet

    • Creating Email ID
    • Sending & Receiving Emails
    • Chatting
    • Searching for Information
    • Downloading

  • Case Study

    • A real-time case study will be provided to students to help them gain confidence in managing office work independently using computers.
